# FireWall-with-Netfliter
基于Netfilter的Linux状态检测防火墙，支持NAT（ 华中科技大学2023学年网络安全课程设计项目，参考https://github.com/RicheyJang/RJFireWall ）
- 可以直接**把kernel_mod文件夹替换到RJFireWall中**，使用它的前端即可(绝对不是自己不想写Readme)
- 具体改动见文档PDF，希望不要完全借鉴(怕查重)
